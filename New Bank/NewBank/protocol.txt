This document details the protocol for interacting with the NewBank server.  

A customer enters the command below and sees the messages returned 

SHOWMYACCOUNTS
Returns a list of all the customers accounts along with their current balance 
e.g. Main: 1000.0 

NEWACCOUNT <Name>
e.g. NEWACCOUNT Savings
Returns SUCCESS or FAIL

MOVE <Amount> <From> <To>
e.g. MOVE 100 Main Savings 
Returns SUCCESS or FAIL

PAY <Person/Company> <Amount> <FromAccount> <ToAccount> <DebitBank> <CreditBank>
e.g. PAY John 100 Main Savings bank3 bank1
Returns SUCCESS or FAIL

BORROWMICROLOAN <LENDER> <Amount> <Income> <Term>
eg:BORROWMICROLOAN Microlender1 123 12 2
Returns LOAN GRANTED/FAIL

REGISTERLENDER <LENDER>
eg:REGISTERLENDER Microlender14
Returns SUCCESS or FAIL

SHOWMICROLENDERS
eg: SHOWMICROLENDERS
Returns LIST OF MICROLENDERS
